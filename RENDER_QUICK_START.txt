1. Click on your new database

--------------------------------------

================================================================================
TROUBLESHOOTING
================================================================================
Problem: Connection timeout
Solution: Check internet connection, verify database is "Available"

Problem: "relation already exists" warnings
Solution: Normal - ignore these (schema uses IF NOT EXISTS)

Problem: Can't find backup
Solution: Run scripts\backup_database.bat first

================================================================================
FUTURE MIGRATIONS
================================================================================
When you need to switch hosting again:

1. pg_dump "old-host-url" > backups/migration.sql
2. scripts\deploy_to_render.bat (new host)
3. psql "new-host-url" < backups/migration.sql
4. Update .env (one time)

NEVER manually recreate tables again!
