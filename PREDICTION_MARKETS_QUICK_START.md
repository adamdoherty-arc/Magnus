### Confidence Score

```
65-69%  = Slight edge
70-79%  = Good edge
80-89%  = Strong edge

```

### Opportunity Score
```
0-50    = Poor opportunity
51-80   = Moderate opportunity
81-100  = Good opportunity
101-120 = Great opportunity
121-150 = Excellent opportunity
```

### Value Rating
```
LOW VALUE      = Avoid (large spread, aligned odds)
MODERATE VALUE = OK (medium spread, some edge)
HIGH VALUE     = Target! (close spread, odds disagreement)
```
